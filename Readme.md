# 📊 HR Analytics Power BI Dashboard

An interactive **HR Analytics Dashboard** built with **Microsoft Power BI** using the **IBM HR Employee Attrition Dataset**. This project analyzes employee attrition, workforce trends, job satisfaction, work-life balance, income distribution, and employee demographics through interactive dashboards.

---

# 📌 Project Overview

This Power BI dashboard provides actionable HR insights by transforming raw employee data into interactive reports. It helps HR professionals monitor employee attrition, identify workforce trends, and support data-driven decision-making.

---

# 🎯 Project Objectives

- Analyze employee attrition across different departments.
- Identify factors affecting employee turnover.
- Monitor workforce demographics.
- Explore employee profiles using interactive filters.
- Build an interactive HR dashboard using Power BI.

---

# 📄 Dashboard Pages

The Executive Dashboard provides a high-level overview of the organization's workforce and key HR metrics.

### Key Metrics

- Total Employees
- Employees Left
- Employees Stayed
- Attrition Rate
- Average Age
- Average Monthly Income
- Average Years at Company

### Visualizations

- Attrition Rate by Department
- Attrition Rate by Job Role
- Attrition by Overtime
- Attrition by Job Satisfaction
- Attrition by Work-Life Balance
- Attrition Trend Analysis

1. Download the repository.

3. Explore the dashboards using slicers and filters.
4. Use the Drill-through feature to navigate to Employee Details.
